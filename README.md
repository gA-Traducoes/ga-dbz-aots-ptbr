## Como Aplicar o Patch

1. Acesse o "Rom Patcher JS" (https://www.marcrobledo.com/RomPatcher.js/).
2. Em ROM file, selecione a ROM original ".nds".
3. Em Patch file, selecione o arquivo ".bps".
4. Clique em Apply patch.
5. Abra a nova ROM gerada no seu emulador.

Funciona no celular e no computador.

## Erros comuns

- **Source ROM mismatch / checksum incorreto:** a ROM base não é exatamente a versão exigida.
- **O jogo não inicia:** aplique novamente sobre uma cópia limpa, sem patch anterior.
- **Arquivo de saída igual ao original:** confirme que selecionou o patch BPS correto e salvou em outro arquivo.
